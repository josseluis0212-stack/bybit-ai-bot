# Comandos Útiles

## Git
```bash
# Commit y push
git add . && git commit -m "mensaje" && git push origin main

# Force push (sobrescribir remoto)
git push origin main --force

# Pull con rebase
git pull origin main --rebase

#Abort merge/rebase
git merge --abort
git rebase --abort
```
