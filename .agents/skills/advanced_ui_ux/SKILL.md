# Skill: Advanced UI/UX Design

Esta skill se enfoca en la creación de experiencias visuales que impacten al usuario y faciliten la toma de decisiones rápidas en trading.

## 🎨 Principios de Diseño "Antigravity"

### 1. Glassmorphism & Depth
- Uso de fondos semitransparentes con `backdrop-filter: blur()`.
- Bordes finos y sutiles (1px) con degradados (Linear Gradients).
- Sombras de neón (`box-shadow`) para resaltar elementos activos.

### 2. Micro-Interacciones
- Transiciones de escala al hacer clic (`active:scale-95`).
- Animaciones de hover para botones suaves.
- Feedback visual instantáneo para cada acción del usuario.

### 3. Visualización de Datos (Real-time)
- Dashboards responsivos que priorizan la información crítica (PnL, Equity, Regime).
- Uso de bibliotecas ligeras (Lucide Icons, Tailwind CSS) para asegurar carga rápida en móviles.

## 🛠️ Herramientas y Frameworks
- **Tailwind CSS**: Para prototipado rápido y sistema de diseño cohesivo.
- **Lucide Icons**: Para iconografía vectorial limpia y moderna.
- **Google Fonts (Outfit/Inter)**: Tipografía legible y futurista.
