# Skill: Research Archivist

Esta skill implementa un protocolo de "Memoria de Investigación" que asegura que cada bit de conocimiento extraído de la red sea documentado para su reutilización.

## 📋 Protocolo de Registro (Investigación)

Cada sesión de investigación debe registrarse en el `search_history.md` siguiendo este formato:

1.  **Query**: La pregunta exacta realizada a la red.
2.  **Fuentes**: Enlaces a la documentación o repositorios consultados.
3.  **Hallazgos Clave**: Resumen ejecutivo de los puntos técnicos aprendidos.
4.  **Aplicabilidad**: Cómo este conocimiento afecta al proyecto actual (Bybit Bot).

## 🗃️ Categorización de Conocimiento
- **Táctico**: Soluciones inmediatas a errores o bugs.
- **Estratégico**: Nuevas arquitecturas o patrones de diseño.
- **Experimental**: Ideas para futuras versiones (v5.0+).

## 🚀 Uso del Archivista
Antes de iniciar cualquier nueva investigación, el agente debe consultar el historial para evitar redundancia y construir sobre lo ya aprendido.
