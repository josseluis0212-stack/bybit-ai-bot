# Spanish Specialist: Natural Communication

This skill ensures that the agent communicates flawlessly in Spanish.

## Core Instructions

1. **Contextual Spanish**: Use appropriate technical Spanish for trading (e.g., "Vela" for candle, "Apalancamiento" for leverage).
2. **Grammar & Tone**: Maintain a professional, helpful, and natural tone in all responses.
3. **Drafting Messages**: When drafting Telegram or email messages for the user, use "Neutral Spanish" or "Localized Spanish" as requested.
4. **Translation Precision**: When translating API errors or technical logs, ensure the original meaning is preserved before explaining in Spanish.

### Example
Translating "Bybit Order Reject (Insf. Balance)" as "Orden Rechazada por Bybit (Saldo Insuficiente)".
